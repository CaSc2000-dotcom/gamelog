# GameLog

GameLog is your gaming journal founded on the principles of *Intentional Gaming*; approaching the hobby with deliberate structure and self-awareness. The idea is to ensure that play remains a meaningful, energizing experience rather than a mindless habit. GameLog seeks to promote a healthy approach to gaming, allowing you a dedicated space to reflect on your game sessions while providing a beautiful and streamlined interface.

## Tech Stack

Next.js (App Router) + TypeScript + Tailwind + shadcn/ui, on Vercel, backed by Supabase (Postgres + Auth + Storage), CI on GitHub Actions

## Local-Run Instructions

Clone the repo, open the folder with your favorite terminal, and run `npm run dev`.
